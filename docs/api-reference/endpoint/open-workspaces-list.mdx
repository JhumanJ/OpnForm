# Workspaces (Public API)

## List Workspaces

```http
GET /open/workspaces HTTP/1.1
Authorization: Bearer <token with `workspaces:read`>
```

## Create Workspace

```http
POST /open/workspaces HTTP/1.1
Authorization: Bearer <token with `workspaces:write`>
Content-Type: application/json

{ "name": "My Workspace" }
```

## Update Workspace

```http
PUT /open/workspaces/{workspaceId} HTTP/1.1
Authorization: Bearer <token with `workspaces:write`>
```

## Delete Workspace

```http
DELETE /open/workspaces/{workspaceId} HTTP/1.1
Authorization: Bearer <token with `workspaces:write`>
```

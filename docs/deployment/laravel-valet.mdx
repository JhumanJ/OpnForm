# Using Laravel Valet

This guide explains how to set up OpnForm locally using Laravel Valet, which is useful for development purposes.

1. Clone the repository and install dependencies:

   ```bash
   git clone git@github.com:JhumanJ/OpnForm.git && cd OpnForm
   composer install
   cd client && npm install
   ```

2. Compile assets:

   ```bash
   npm run dev # or build
   ```

3. Configure Laravel:

   - Create `.env` file: `cp .env.example .env`
   - Configure database settings in `.env`
   - Run Artisan commands:
     ```bash
     php artisan key:generate
     php artisan jwt:secret
     php artisan migrate
     ```

4. Set up S3 bucket (or equivalent) and configure AWS environment variables.

For more detailed instructions, including S3 bucket setup and CORS configuration, please refer to our full Laravel Valet deployment guide.

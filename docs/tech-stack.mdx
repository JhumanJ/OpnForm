# Tech Stack

OpnForm is built using a modern and robust technology stack, ensuring high performance, scalability, and maintainability. Here's an overview of the main technologies used in the project:

## Backend

- **[PHP](https://www.php.net/)**: Version 8.0 or higher
- **[Laravel](https://laravel.com/)**: The PHP framework that powers the backend of OpnForm

## Database

- **[MySQL](https://www.mysql.com/)/[MariaDB](https://mariadb.org/)** or **[PostgreSQL](https://www.postgresql.org/)**: OpnForm supports both of these popular relational database management systems

## Frontend

- **[Vue.js 3](https://v3.vuejs.org/)**: The progressive JavaScript framework used for building the user interface
- **[Nuxt.js](https://nuxt.com/)**: The Vue.js framework that enables server-side rendering and other powerful features
- **[TailwindCSS](https://tailwindcss.com/)**: A utility-first CSS framework for rapidly building custom user interfaces

## Asset Compilation

- **[Node.js](https://nodejs.org/)**: Used for running JavaScript tools in the development environment
- **[NPM](https://www.npmjs.com/)/[Yarn](https://yarnpkg.com/)**: Package managers for installing and managing frontend dependencies

## Additional Technologies

- **[Docker](https://www.docker.com/)**: Containerization platform used for easy deployment and development setup
- **[Redis](https://redis.io/)**: In-memory data structure store, used for caching and as a message broker
- **[AWS S3](https://aws.amazon.com/s3/)** (or compatible): Used for file storage

This tech stack allows OpnForm to deliver a fast, responsive, and feature-rich form building experience. For detailed installation instructions and system requirements, please refer to our [Docker deployment guide](deployment/docker) or [Laravel Valet deployment guide](deployment/laravel-valet).
